# Four H Group Website

A modern, responsive corporate website inspired by the Four H Group design.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Navigation**: Sticky header with smooth scrolling
- **Modern UI**: Clean, professional design with animations
- **Key Sections**:
  - Hero section with call-to-action
  - Awards banner
  - Managing Director's message
  - Company information with statistics
  - Product gallery
  - Contact form
  - Footer with quick links

## Structure

```
Four-H-Group/
├── index.html      # Main HTML file
├── styles.css      # Styling and responsive design
├── script.js       # Interactive features
└── README.md       # This file
```

## How to Use

1. Open `index.html` in your web browser
2. Navigate through sections using the top menu
3. The website is fully responsive - try resizing your browser window

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #004080;
    --secondary-color: #0066cc;
    --accent-color: #00a86b;
}
```

### Content
- Edit text in `index.html`
- Add your own images to an `images/` folder
- Update company information and statistics
